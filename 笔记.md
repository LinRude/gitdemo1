2. git日常操作 （日常工作流）

- git init (初始化一个仓库)

- git add index.html(将需要的东西放入暂存区中)

- git commit(将暂存区的东西形成一个版本)


npm init 初始化

npm add express 添加node_modules

3. git status   查看暂存区

4. git rm -r --cached ./node_modules/   把node_modules移出暂存区（不需要这个）

5. git commit -m "安装依赖"
